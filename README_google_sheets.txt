4,320N お世話ゲーム：Googleスプレッドシート自動保存版

1. Googleドライブで新しいスプレッドシートを作ります。
   例：4320N_お世話ゲーム_行動ログ

2. スプレッドシートで「拡張機能」→「Apps Script」を開きます。

3. google_apps_script/Code.gs の中身を、Apps Script の Code.gs に貼り付けます。

4. Apps Script 画面で setup を一度実行します。
   初回は権限確認が出ます。
   これで logs シートと1行目の見出しが作られます。

5. 「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」を選びます。
   実行するユーザー：自分
   アクセスできるユーザー：全員
   でデプロイします。

6. 表示されたウェブアプリURLをコピーします。

7. index.html の上の方にある
   const LOG_ENDPOINT = "";
   の "" の中に、コピーしたURLを貼ります。

8. GitHub Pagesにアップロードします。
   参加者が操作すると、logs シートに1行ずつ記録されます。

注意
・研究IDで追跡します。氏名、学校名、住所などは入れません。
・GoogleフォームURLは index.html の SURVEY_URL を差し替えてください。
・動作確認中は DEV_MODE = true のままで大丈夫です。本調査では false にします。
